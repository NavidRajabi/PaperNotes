Segmentation from Natural Language Expressions

In this paper, they address the following problem:

given an image and a natural language expression that describes a certain part of the image, we want to segment the corresponding region(s) that covers the visual entities described by the expression

objective function

\[\begin{aligned} Loss &= \frac{1}{WH}\sum_{i=1}^W\sum_{j=1}^HL(v_{ij},M_{ij})\\ L(v_{ij},M_{ij})&=\begin{cases}\alpha_f \log(1+\exp (-v_{ij})), \text{ if } M_{ij}=1\\ \alpha_b \log (1+\exp (v_{ij})), \text{ if } M_{ij}=0\end{cases} \end{aligned} \]
